# NetGuard Pro

**Suite de cybersecurite professionnelle pour Windows**

NetGuard Pro protege votre systeme avec 4 modules essentiels :
- **IDS/IPS** — Detection d'intrusion en temps reel
- **Antivirus** — Scan ClamAV, YARA, VirusTotal
- **Firewall** — Gestion intelligente du pare-feu Windows
- **Network Mapper** — Carte mondiale des menaces
